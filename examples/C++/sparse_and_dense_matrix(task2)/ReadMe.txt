В matrix.cpp и matrix.hpp описан интерфейс для работы с матрицей и два класса - различных представления:
IterableDenseMatrix и IterableSparseMatrix. 


Если методы get и set, позволяющие получать и изменять элементы и методы, возвращающие константные итераторы для прохода по строкам или столбцам. 


Элемент матрицы, возвращаемый итератором содержит в себе номер строки, столбца и само значение элемента.


В качестве функций для графа, реализованы: 

1) поиск вершины/вершин с наибольшим количеством входящих и исходящих ребер

2) поиск количества компонент связности

Функции принимают граф в виде указателя на базовый класс.
