Индексация матриц matrix(n,m) идет от 0 до n-1 и от 0 до m-1 соответственно. 

Структура "Triplet" - список для хранения элементов матрицы.
Имеет два поля - индексы матрицы, значение элемента и ссылки на следующий и предыдущий элементы. 

Класс "Row" - вспомогательный класс для основного класса "SparseMatrix". 
Он нужен в первую очередь для того, чтобы перегрузить операторы адресации. 

Класс "SparseMatrix" - основной класс, задающий матрицу. 
Его методы:

Вспомогательные: 
1) handle_exception_out_of_range
2) handle_exception_incorrect_size
3) handle_wrong_size_assignment

Эти 3 метода помогают выводить ошибки при выполнении операций с матрицами. 

4) find_elem - ищет элемент матрицы с заданными номерами и возвращает ссылку на него. 
Если такого нет, создастся нулевой и вернется на него ссылка. 
5) get_link - достает первый индекс, получает второй и вызывает find_elem
6) find - возвращает указатель на элемент, если такой есть, иначе nullptr
7) add_elem - вызывается из find_elem, если нужно создать элемент
8) tf - проверяет две матрицы на равенство, возвращает true в случае полного совпадения, иначе false
9) set_zero_rows_cols - заполняет массивы, показывающие, какие строки или столбцы являются нулевыми

Методы, доступные пользователю: 
1) SparseMatrix(int a, int b) - конструктор с двумя и более параметрами
2) SparseMatrix(const SparseMatrix & m) - конструктор копирования
3) operator= - присваивание матриц
4) operator[] - обработчик индексной адресации, получает первый индекс и записывает его, возвращает Row &
5) show_matrix - вывод элементов матрицы
6) operator* - умножение матриц
7) operator+(const SparseMatrix & m) - сложение матриц
8) operator+(int x) - обработчик адресной арифметики, получает первый индекс и записывает его, возвращает Row &
9) operator==, operator!= - сравнивают матрицы
10) get - достает значение элемента по двум индексам
11) set - устанавливает значение в указанный элемент
12) num_rows, num_cols - возвращают соответственно количество строк и столбцов
13) ~SparseMatrix - деструктор, очищающий память
