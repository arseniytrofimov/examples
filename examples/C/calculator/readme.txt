Задание состояло в том, чтобы написать калькулятор, работающий в том числе с переменными.
Описание программы.



Программа работает с целыми неотрицательными числами и операциями +, -, /, *
Функции push_stack, pop_stack и free_stack работают со стеком.

Структура variable содержит в себе название переменной и ее значение, а
table - массив таких структур - своего рода таблица переменных со значениями

Функции типа handle_..._exception обрабатывают ошибки ввода
Рассмотренные ошибки:
-неверный балланс скобок с указанием, каких больше
-пустые скобки
-пропущенное значение или операция
-неверные переменные
-пропущенная операция между скобкой и значением
-невалидный символ
-целое число, большее, чем максинт

Все происходит через список
В него добавляются значения функциями add_to_list_variable,
add_to_list_number, add_to_list_operation
Переменная хранится как ее номер в таблице переменных

Функция check_propietary помогает отследить ошибки ввода
Функция get_full_number досчитывает число, если встретилась некоторая цифра
Функция get_full_variable досчитывает переменную, если встретилась буква, 
и сразу добавляет ее в таблицу с помощью функции add_variable

Функция get_next_char получает следующий символ, минуя пробелы

Функция descent запускает рекурсивный спуск, состоящий из трех функций,
работающих соответственно с +,-, с *,/, с переменными и числами
Эти функции begin_descent, mult_div_descent, symbol_descent

Функция add_values получает у пользователя значения для переменных
Если не все переменные заполнены, выдаст ошибку

Функция result считает конечный результат через стек.
