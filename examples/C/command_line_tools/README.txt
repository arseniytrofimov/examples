Задание состояло в том, чтобы реализовать некоторые функции командной строки.

Скрипт script.sh и папки dir1,dir2,dir3 - тесты

------------------------------------------------------------------

Реализуемые утилиты:

   cp.c file1 file2

Создает file2 - копию file1 с теми же правами доступа.
Если file2 уже существует, будет выдана соответствующая ошибка. 
Если невозможно открыть file1, будет выдана соответствующая ошибка.
Если аргументов больше или меньше двух, также будет выдана ошибка.

   cmp.c file1 file2

Сравнивает посимвольно два файла.
Если они идентичны, выведет соответствующее сообщение.
Если найдется различие, то будет выведено сообщение, в какой строке и в каком символе отличие.
Если невозможно открыть какой-то из файлов - будет выведена соответствующая ошибка.
Если аргументов больше или меньше двух, также будет выдана ошибка.

   cat.c [-n] files

Выводит в консоль содержимое файлов.
Если указан флаг -n, то строки выведенных файлов будут пронумерованы.
Если не было указано ни одного файла, будет выведена соответствующая ошибка.
Если невозможно открыть файл, также будет выдана ошибка.
